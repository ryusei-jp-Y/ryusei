「どさいても、庄内」Webページ
===============
## 概要
インターンシップで、「Withコロナの時代に沿った新しい観光のカタチ」というテーマに取り組んだ際に作成したWebページです。提案する観光ツアーの概要やオンラインツアー風の動画を視聴することができます。

## 構成
本作品のファイル構成は以下の通りです。
```
intern-web
├── README.md
├── shonai.html
├── main.css
├── image
│   ├── back_image.PNG
│   ├── form2.pmg
│   ├── thumbnail_turu.png
│   └── tour_form.PNG
└── movie
    └── intern_sample.mp4
```

## 使用言語等
本作品を作成するにあたり使用した言語、技術は以下の通りです。
* HTML5
* CSS3
* 動画編集
